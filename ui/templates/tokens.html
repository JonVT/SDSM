Chat Token Reference

Use these tokens in Chat and the Welcome Message. They are replaced automatically before sending to the server.

Available Tokens

Token Description
[ServerName] The server name
[WorldName] The world name
[WorldID] The world ID
[StartLocation] Start location
[StartCondition] Start condition
[Date] Current date (YYYY-MM-DD)
[Time] Current time (HH:MM:SS)
[LastPlayer] Name of the last player to connect

Examples


Example 1: Welcome Message
"Welcome to [ServerName]! World: [WorldName]."

Example 2: Chat
"/say Hello [LastPlayer], you joined at [Time] on [Date]."

Tokens are case-insensitive and work anywhere messages are sent via the UI (Chat and Welcome Message).
